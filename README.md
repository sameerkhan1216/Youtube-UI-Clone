# Youtube-UI-Clone
A responsive YouTube UI clone built with JavaScript, HTML, and CSS. Features a dynamic video grid, collapsible sidebar, and functional search bar to replicate the modern YouTube user experience


📺 YouTube UI Clone
A sleek, responsive replica of the YouTube homepage designed to showcase modern front-end development skills using pure JavaScript.
🌟 Key Features
Dynamic Video Grid: Responsive layout that adjusts video cards based on screen size using CSS Grid and Flexbox.
Collapsible Sidebar: A functional hamburger menu that toggles the sidebar for improved navigation.
Interactive Search Bar: Includes a styled search container with hover effects and focus states.
Category Filters: Scrollable category tabs at the top to filter content (UI only).
Dark/Light Mode: Toggle between themes for a personalized viewing experience.
🛠️ Tech Stack
Languages: HTML5, CSS3, JavaScript (ES6+).
Layout: CSS Grid, Flexbox, and Media Queries for full responsiveness.
Icons: Integrated with Material Icons or Font Awesome.
